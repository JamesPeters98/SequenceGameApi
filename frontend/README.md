# React + TypeScript + Vite + shadcn/ui

This is a template for a new Vite project with React, TypeScript, and shadcn/ui.

## Environment variables

- `VITE_API_BEARER_TOKEN` (optional): when set, the frontend includes
  `Authorization: Bearer <token>` on API requests.
- Leave this unset for environments where the backend does not require
  bearer auth (for example production).
